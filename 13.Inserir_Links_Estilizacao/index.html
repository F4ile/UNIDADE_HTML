Inernet
Rede mundial de computadores formada por redes interligadas.

HTML
Hyper Text Markup Language é uma linguagem de marcação de Hiper texto.
É a linguagem de marcação padrão para criaçao de páginas web.

Desenvolvimento Web

Desenvolvimento Web
É o processo de criação e manutenção de websites e aplicações que são acessado através da internet.
Word Wide Web (www)
